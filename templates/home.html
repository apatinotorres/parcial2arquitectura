Sistema de control de acceso vehicular - RestAPI

Base url:

Vehiculos

/carros : Metodo: GET

Obtiene un json con todos los vehiculos registrados por la API


/carros/<placa> : Metodo: PUT

Modifica un vehiculo segun su placa que este registrado por la API. Ejemplo de json que se envia desde el cliente:

{ "codigo": "T00056470", "fecha": "18/04/22 9:55:19", "placa": "qhg984" }

/carros : Metodo: POST

Crea un vehiculo nuevo segun los atributos que mande el cliente y posteriormente se registra por la API. Ejemplo de json que se envia desde el cliente:

{ "codigo": "T00056470", "fecha": "18/04/22 9:55:19", "placa": "qhg984" }

/carros/<placa> : Metodo: DELETE

Elemina un vehiculo segun su placa que este registrado por la API.

Usuarios

/usuarios : Metodo: GET

Obtiene un json con todos los usuarios registrados por la API


/usuarios/<id> : Metodo: PUT

Modifica a un usuario segun la id suminisitrada. Ejemplo de json que debe ser enviado:

{ "codigo": "T00056470", "nombres": "Antonio Jose Patiño Torres", "rol": "estudiante", "edad":"21" }

/usuarios : Metodo: POST

Crea un usuario nuevo segun los datos suminisitrados por el cliente. Ejemplo de json que debe ser enviado:

{ "codigo": "T00056470", "nombres": "Antonio Jose Patiño Torres", "rol": "estudiante", "edad":"21" }

/usuarios/<id> : Metodo: DELETE

Elimina a un usuario segun la id suminisitrada.
